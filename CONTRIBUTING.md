# Contributing to AkhAI

After seed funding, we plan to:
- Open selected components for community contribution
- Establish contribution guidelines
- Create a contributor license agreement

## Feedback Welcome

While we're not accepting code contributions yet, feedback is valuable:
- **Bug reports**: Open an issue with reproduction steps
- **Feature suggestions**: Use GitHub Discussions
- **Questions**: Check existing issues or open a new one
